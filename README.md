# QBASIC
基于 Qt 框架实现的简易 BASIC 语言解释器，支持经典 BASIC 核心语法与语句执行，适配跨平台桌面应用，适合 Qt 学习与编程语言设计实践展示。

## 项目介绍
本项目是一款基于 C++ 和 Qt 开发的 BASIC 语言解释器，还原经典 BASIC 语言的核心语法体系（如编号语句、输入输出、变量运算、注释等），
通过 Qt 构建可视化交互界面，实现 BASIC 程序的编辑、编译、运行与结果展示功能。
项目完整实现了 BASIC 程序的解析、语义分析与执行逻辑，代码结构清晰、注释完善，
既适合 Qt 初学者学习界面开发与事件处理，也可作为编程语言解释器设计的入门案例。

## 技术栈
- 编程语言：C++
- 开发框架：Qt 6（Qt Widgets）
- 核心技术：Qt 文本编辑（QTextEdit）、信号与槽机制、自定义语法解析器、语句执行器、界面布局（QVBoxLayout/QHBoxLayout）、事件处理
- 开发工具：Qt Creator、Git
- 运行环境：跨平台支持（Windows/macOS/Linux）

## 核心功能
- 经典 BASIC 语法支持：解析执行带编号的 BASIC 语句，包含 REM 注释、INPUT 输入、LET 变量赋值、PRINT 输出、END 程序结束等核心语句
- 可视化交互界面：左侧编辑区支持 BASIC 程序代码编写，右侧运行区实时展示程序输出结果，操作流程清晰
- 程序执行控制：支持单步执行、完整运行、重置程序等操作，实时反馈执行状态（如语法错误提示、变量值变化）
- 变量管理：支持整数 / 数值型变量定义与运算，保留 BASIC 变量使用规则，支持运行中修改变量值
- 语法基础高亮：对 BASIC 关键字（如 INPUT/PRINT/LET）、注释、编号语句进行基础语法高亮，提升代码可读性
- 跨平台兼容：基于 Qt 框架编译运行，可在 Windows、macOS、Linux 系统上正常使用，适配不同屏幕分辨率

## 示例 BASIC 程序（可直接测试）
10 REM 计算两数之和
20 INPUT n1
30 INPUT n2
40 LET total = n1 + n2
50 PRINT total
60 END

## 开发亮点
- 模块化设计：将语法解析、语句执行、界面展示拆分为独立模块，代码解耦、易于维护与扩展
- 语法解析优化：逐行解析 BASIC 编号语句，实现基础语法校验与错误提示，提升程序鲁棒性
- Qt 界面实践：熟练运用 Qt 布局、信号槽、文本编辑组件，实现直观易用的交互界面
- 跨平台特性：依托 Qt 框架的跨平台优势，无需修改核心代码即可适配多操作系统
- 注释完整：核心逻辑模块附带详细注释，便于学习与二次开发
